# Real Estate Price Prediction App

This is a Streamlit web application for real estate investment and price prediction using Machine Learning models.

## Features
- Predict property investment quality
- Price prediction using XGBoost model
- Interactive UI built with Streamlit

## Tech Stack
- Python 3.10
- Streamlit
- Scikit-learn
- XGBoost
- NumPy
- Pandas

## How it works
The app loads pre-trained ML pipelines stored as `.joblib` files and performs predictions based on user input.

## Deployment
This app is deployed using **Streamlit Cloud**.
